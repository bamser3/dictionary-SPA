# Dictionary SPA

A simple web based dictionary app that allows you to search for English words, view their definitions, examples, synonyms, antonyms, and listen to pronunciation.

I used the [Free Dictionary API](https://dictionaryapi.dev/) for the words.

---

## Features

* Search for any English word
* Display the **word**, **part of speech**, **definition**, and **example usage**
* Shows synonyms and antonyms when available
* Pronunciation audio playback

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/dictionary-app.git
cd dictionary-app
```

2. Open `index.html` in your browser.

---

## Usage

1. Type a word in the search input.
2. Press **Enter** or click the **Search** button.
3. The app will display:

   * Word and part of speech
   * Definition
   * Example sentence
   * Synonyms / Antonyms
   * Pronunciation button to play audio
4. If the word is not found, a "Not found" message is displayed.
